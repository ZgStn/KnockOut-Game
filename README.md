# KnockOut-Game

A simple dice game built with **HTML, CSS, and JavaScript** during a distance course in **Web Development**.

## About the project

Knockout Game is a small browser-based game where the player first chooses a **knockout number** between 6 and 9.

Then the player rolls two dice:

- If the sum of the dice is **equal to the selected knockout number**, the game ends.
- If the sum is **different**, the sum is added to the player's **total score**.
- The player can continue rolling until the knockout number is hit.
- The game can be restarted at any time with the **Reset the Game** button.

## Learning purpose

This project was created as part of a web development course to practice:

- DOM manipulation

- Event handling

- JavaScript functions

- Conditional logic

- Random number generation

- Updating UI dynamically
